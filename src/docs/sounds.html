Sound

 

  1. Press the Settings button to open the setting form

  2. Press the Sound button to open the file dialog

  3. Locate and open the desired audio file (acc, mp3 or wav)

  4. The selected audio file will be played

  5. In countdown mode, the selected audio file will be played when the time reaches 00:00

  6. To stop the audio, press the Stop, Reset or any of the Preset buttons
