Preset Buttons

 

  1. Press the Settings button to open the setting form

  2. Type in the desired name of the Preset button using the keyboard

  3. Type in the desired minutes and seconds

  4. Repeat this to all the Preset buttons you want to modify

  5. Press the Apply button to commit changes and to return to the main window

  6. To use any of the Presets, just press it and then press Start
